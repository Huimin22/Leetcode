## 3. Key Points

Stack → store opening brackets.

Push → add (, {, [.

Match → check closing bracket against top.

Pop → if match; else invalid.

Empty → stack must be empty at end.


## 4. Code Implementation:


```

class Solution:
    def isValid(self, s: str) -> bool:
        dic = { ')':'(', '}':'{' , ']':'['}
        stack = []
        for c in s:
            if c in dic:
                if not stack or stack.pop()!= dic[c]:                  
                            return False
            else:
                stack.append(c)
        return not stack
   
```

    Time：O(n)
        
    Space: O(1)

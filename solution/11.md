## 3. Key Points

area = (j−i)×min(height[i],height[j])

Use two pointers from both ends, always move the shorter line
area = width × shorter height, to efficiently find the maximum water.

## 4. Code Implementation:


```
   class Solution:
    def maxArea(self, height: List[int]) -> int:
        i, j = 0, len(height) - 1
        res = 0
        while i < j:
            res = max(res, (j-i)*min(height[i], height[j]))
            if height[i] > height[j]:
                j -= 1
            else:
                i += 1
        return res


```

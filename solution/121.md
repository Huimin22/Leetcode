## 4. Code Implementation:

```
code
   class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_p = prices[0]
        max_pro = 0
        for i in range(len(prices)):
            if prices[i] < min_p:
                min_p = prices[i]
            max_pro = max(max_pro, prices[i]-min_p)
        return max_pro

```
